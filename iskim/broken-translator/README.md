# 🌐 Broken Translator (번역기 전화 놀이)

> **쓸모없음 지수: ⭐⭐⭐⭐⭐ (5/5)**

## 무엇을 하는 웹사이트인가요?

이 웹사이트는 여러분의 소중한 문장을 **완벽하게 망가뜨립니다**! 🎉

입력한 문장이 다음과 같은 여정을 거칩니다:
```
한국어 → 핀란드어 → 아랍어 → 일본어 → 다시 한국어
```

결과? 원래 의미와는 전혀 다른 엉뚱한 문장! 마치 전화 놀이처럼요.

## 기술 스택

- **Frontend**: React + Vite
- **Styling**: Tailwind CSS
- **API**: Google Cloud Translation API
- **CI/CD**: GitHub Actions + Vercel

## 로컬 개발

```bash
# Node.js 버전 설정 (nvm 사용 시)
nvm use

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
```
